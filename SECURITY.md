# Security Policy

`latch-core` is a preflight package risk engine. It does not prove that a package is safe or malicious.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting Issues

Do not include tokens, private packages, credentials, or proprietary code in public issues.

When reporting a security concern, include:

- affected package/version,
- relevant report JSON if safe to share,
- reproduction steps,
- Node/npm versions,
- expected and actual behavior.

## Language

Use careful language. Findings are scanner signals and review prompts, not malware verdicts.
